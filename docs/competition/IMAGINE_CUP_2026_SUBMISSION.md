# Imagine Cup 2026 Submission Guide

This document provides a checklist and guide for submitting Dream Flow to Microsoft Imagine Cup 2026.

## Competition Requirements

### Mandatory Requirements

- ✅ **At least 2 Microsoft AI services**: Azure Content Safety + Azure Computer Vision
- ✅ **MVP submission by January 9, 2026**
- ✅ **Pitch deck** (comprehensive presentation)
- ✅ **Pitch video** (3 minutes)
- ✅ **Demo video** (2-3 minutes)
- ✅ **Student eligibility**: 18+ by Sept 30, 2025, enrolled Sept 30, 2025 - May 31, 2026

## Submission Checklist

### 1. Registration

- [ ] Register team on [Imagine Cup website](https://imaginecup.microsoft.com/)
- [ ] Verify all team members meet eligibility requirements
- [ ] Access $5,000 Azure credits (provided upon registration)
- [ ] Confirm team name and details

### 2. Technical Requirements

- [ ] Azure Content Safety integrated and working
- [ ] Azure Computer Vision integrated and working
- [ ] Application deployed to Azure (or deployment-ready)
- [ ] All Azure services tested and verified
- [ ] Demo environment configured and tested

### 3. Submission Materials

#### Pitch Deck

- [ ] Problem statement clearly defined
- [ ] Solution overview with on-device AI + Azure services hybrid approach
- [ ] Technology stack highlighted (local models + Azure AI)
- [ ] Market opportunity (Education + Health categories)
- [ ] Business model (subscription tiers)
- [ ] Team introduction
- [ ] Impact and social good (accessibility, child safety, privacy)
- [ ] Competitive advantage (privacy-first, offline-capable)
- [ ] Roadmap and vision
- [ ] Ask/Support needed
- [ ] Saved as PowerPoint format (`.pptx`)

#### Pitch Video (3 minutes)

- [ ] Team members introduce themselves
- [ ] Problem clearly explained
- [ ] Solution demonstrated
- [ ] Azure AI services highlighted
- [ ] Impact and market opportunity discussed
- [ ] Professional recording quality
- [ ] Clear audio
- [ ] Saved as MP4 format

#### Demo Video (2-3 minutes)

- [ ] App walkthrough (mobile app)
- [ ] On-device story generation demonstrated
- [ ] Azure Content Safety moderation shown
- [ ] Azure Computer Vision image analysis shown
- [ ] Personalization features
- [ ] Family mode showcase
- [ ] Multi-platform capabilities
- [ ] High quality screen recordings
- [ ] Clear voiceover
- [ ] Saved as MP4 format

### 4. Documentation

- [ ] Technical architecture document (`docs/competition/TECHNICAL_ARCHITECTURE.md`)
- [ ] Business plan document (`docs/competition/BUSINESS_PLAN.md`)
- [ ] Azure integration documentation (`docs/AZURE_INTEGRATION.md`)
- [ ] Deployment guide (`docs/DEPLOYMENT_AZURE.md`)
- [ ] README updated with Azure badges and services

### 5. Code Repository

- [ ] All code committed to GitHub
- [ ] Repository is public or accessible to judges
- [ ] README.md is comprehensive and up-to-date
- [ ] Azure deployment files included (`azure/` directory)
- [ ] GitHub Actions workflow configured (`.github/workflows/deploy-azure-app-service.yml`)
- [ ] All environment variable templates documented

### 6. Testing

- [ ] Azure Content Safety tested with sample content
- [ ] Azure Computer Vision tested with sample images
- [ ] End-to-end story generation tested
- [ ] Demo scenarios tested and documented
- [ ] Performance benchmarks recorded
- [ ] Error handling verified

### 7. Pre-Submission Review

- [ ] All submission materials reviewed by team
- [ ] Pitch deck reviewed by mentors/advisors (if available)
- [ ] Demo video tested on different devices
- [ ] All links in materials are working
- [ ] All Azure services are accessible
- [ ] Demo environment is stable

## Submission Process

### Step 1: Prepare Materials

1. **Organize Files**:

   - Create `docs/competition/submission/` directory
   - Place pitch deck, videos, and documentation here

2. **Final Review**:
   - Check all files are complete
   - Verify video quality and file sizes
   - Ensure all links work

### Step 2: Upload to Imagine Cup Portal

1. Log into [Imagine Cup portal](https://imaginecup.microsoft.com/)
2. Navigate to submission section
3. Upload materials:
   - Pitch deck (PowerPoint)
   - Pitch video (MP4)
   - Demo video (MP4)
4. Fill out submission form:
   - Project description
   - Technology stack
   - Azure services used
   - Category (Education/Health)

### Step 3: Submit

1. Review all uploaded materials
2. Verify all required fields are completed
3. Submit before deadline (January 9, 2026, 11:59 PM UTC)
4. Save confirmation email/receipt
