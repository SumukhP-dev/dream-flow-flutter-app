# Stripe Product Image Requirements

## Overview

Stripe requires product images for each subscription tier. These images appear at checkout and help customers understand what they're purchasing.

## Image Specifications

- **Format**: JPEG, PNG, or WEBP
- **Size**: Under 2MB
- **Recommended Dimensions**: 512x512px or larger (square format works best)
- **Aspect Ratio**: 1:1 (square) is recommended
- **Quality**: High resolution for clarity

## Required Images

### 1. Premium Subscription Image

**Purpose**: Represent the Premium tier (used for both Monthly and Annual)

**Content Suggestions**:
- App icon/logo with "Premium" badge
- Star or crown icon to indicate premium status
- Clean, professional design
- Brand colors consistent with app

**Text Elements** (if included):
- "Premium" or "Premium Plan"
- Optional: Key features (Unlimited Stories, All Themes, Offline Mode)

**Design Style**:
- Modern, clean, family-friendly
- Professional but approachable
- Consistent with app branding

---

### 2. Premium Annual Badge Variant (Optional)

**Purpose**: Same as Premium but with "Annual" or "Save 33%" indicator

**Content Suggestions**:
- Same base design as Premium image
- Add "Annual" text or badge
- Include "Save 33%" or "$6.67/month" text
- Visual indicator of savings (e.g., discount badge)

---

### 3. Family Subscription Image

**Purpose**: Represent the Family tier (used for both Monthly and Annual)

**Content Suggestions**:
- Multiple user icons or family silhouette
- "Family" text or badge
- Up to 5 family members indicator
- Family-friendly imagery (parents + children)

**Text Elements** (if included):
- "Family Plan" or "Family"
- "Up to 5 Members"
- Optional: Key features (Child Profiles, Family Analytics)

**Design Style**:
- Warm, inclusive, family-oriented
- Professional but welcoming
- Emphasizes multi-user capability

---

### 4. Family Annual Badge Variant (Optional)

**Purpose**: Same as Family but with "Annual" or "Save 33%" indicator

**Content Suggestions**:
- Same base design as Family image
- Add "Annual" text or badge
- Include "Save 33%" or "$10/month" text
- Visual indicator of savings

---

## Quick Design Checklist

For each image:

- [ ] Square format (1:1 aspect ratio)
- [ ] Minimum 512x512px resolution
- [ ] File size under 2MB
- [ ] High quality/clear
- [ ] Brand colors consistent
- [ ] Text is readable (if included)
- [ ] Professional appearance
- [ ] Family-friendly content
- [ ] Saved as PNG or JPEG

---

## File Naming Convention

Suggested file names:
- `premium-subscription.png`
- `premium-annual.png` (optional variant)
- `family-subscription.png`
- `family-annual.png` (optional variant)

---

## Alternative: Use App Assets

If you already have app assets, you can:

1. **Use App Icon**: Resize your app icon to 512x512px
2. **Create Badge Overlay**: Add "Premium" or "Family" text overlay
3. **Use Screenshots**: Crop app screenshots to square format (less recommended)

---

## Testing

Before uploading to Stripe:

1. **Preview at different sizes**: Check how it looks at checkout
2. **Test on mobile**: Ensure it looks good on small screens
3. **Verify file size**: Confirm under 2MB
4. **Check format**: Ensure JPEG, PNG, or WEBP

---

## Notes

- Images are optional but recommended for better conversion
- If you don't have images ready, you can create products first and add images later
- Stripe will use a default placeholder if no image is provided
- You can update images anytime in the Stripe Dashboard
