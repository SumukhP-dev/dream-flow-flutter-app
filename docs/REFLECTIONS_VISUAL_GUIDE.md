## Key Interactions

1. **Homepage Widget Tap**: Opens full reflections screen
2. **My Stories Sparkle Icon**: Opens full reflections screen
3. **Tab Navigation**: Switch between Overview/History/Insights
4. **Reflection Entry Tap**: Could expand to show full details (future)
5. **Empty State CTA**: Navigates back to create story
6. **Pull to Refresh**: Reloads reflection data (in History tab)

## Responsive Behavior

- **Portrait**: Single column layout
- **Landscape**: Could show 2-column layout for tablets (future)
- **Small screens**: Scales gracefully with scrolling
- **Large screens**: Max width constraints maintain readability

## Accessibility

- **High contrast**: White text on dark backgrounds
- **Large touch targets**: Buttons and cards are easy to tap
- **Clear labels**: All icons have text labels
- **Semantic structure**: Proper heading hierarchy
- **Screen reader support**: Meaningful descriptions

## Performance Considerations

- **Lazy loading**: Only loads visible items in history list
- **Caching**: Uses SharedPreferences for offline access
- **Pagination**: Limits to 60 most recent reflections
- **Optimistic updates**: Shows data immediately, syncs in background
- **Graceful degradation**: Falls back to local analytics if API fails
